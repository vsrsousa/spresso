# Visual Guide: New GUI Features

This document provides a visual representation of the three new features in the xespresso GUI.

## Overview

The three features are all located in the **Codes Configuration** page:
1. Module Listing (collapsible section at top)
2. QE Version Specification (in Auto-Detect Codes form)
3. Version Selection (in Existing Codes Configuration section)
