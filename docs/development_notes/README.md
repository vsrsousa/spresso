generated during the development of xespresso features.

## Categories

### GUI Documentation
- `GUI_*.md` - GUI implementation notes and improvements
- `VISUAL_*.md` - Visual guides and UI changes

### Implementation Notes
- `IMPLEMENTATION_*.md` - Feature implementation details
- `*_IMPLEMENTATION.md` - Specific feature implementations

### Bug Fixes
- `BUG_FIX_*.md` - Bug fix documentation
- `*_FIX_*.md` - Various fixes and corrections

### Security
- `SECURITY_*.md` - Security-related changes and summaries

### Features
- `CODES_*.md` - Quantum ESPRESSO codes configuration
- `MACHINE_*.md` - Machine configuration
- `PSEUDOPOTENTIAL*.md` - Pseudopotentials handling
- `WORKFLOW_*.md` - Workflow documentation

These files are kept for historical reference and development context.
